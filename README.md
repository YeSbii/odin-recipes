# Odin recipes

This is my very first HTML web development project from **The Odin Project** curriculum.

## 🍳 Featured Recipes

- **Easy Lasagna** - A quick and delicious layered Italian pasta dish.
- **Chicken and Potatoes** - Comfort food made simple.
- **Salad & Eggs** - Healthy and easy-to-make options.
- **Tiramisu Icebox Cake** - A delicious dessert cake.

## 🚀 Skills Learned

- Structuring web pages using fundamental HTML tag (`<h1>`, `<p>`, `<ul>`, `<ol>`).
- Linking multiple web pages together (`<a>`) and embedding images (`<img>`).
- Managing source code and deploying the live website using **Git & Github**.
